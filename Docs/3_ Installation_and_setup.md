# Installation and Setup
<h3>Linux</h3>

- Clone the repository
  - download the zip or
  - download in terminal

    ```bash
        git clone https://github.com/su1nta/FaceReco
    ```

- cd into the project directory

    ```bash
        cd FaceReco
    ```

- run `setup.sh`

    ```bash
        ./setup.sh
    ```
